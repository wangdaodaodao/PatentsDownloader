# PatentsDownloader  

## **运行环境**
+ python 3.6
- 已更新2020.5.23
- 已更新 2022.6.13 修改get方法，实现protobuf方法访问万方数据库页面
    
## **基本思路**：    
##### 1.输入关键词，从万方专利库获得专利号等相关信息；    
##### 2.由专利号在 http://www.drugfuture.com/cnpat/cn_patent.asp 网站内下载pdf专利；
